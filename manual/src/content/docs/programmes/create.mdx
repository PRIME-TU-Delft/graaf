Only super admins can create new graph programmes. If you are a super admin,
you can create a new programme by clicking the "Add Programme" button on
the <a href={import.meta.env.HOST + '/graph-editor/programs'}>Programs page</a>.

When creating a new programme, you will need to provide the following information:

- Programme name
